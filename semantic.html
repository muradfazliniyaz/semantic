Semantics in HTML

👉 In HTML, for example,the ‹h1› element is a semantic element, which gives the text it wraps around the role (or meaning) of “a top level heading on your page.”

▶Here is a top level heading

👉 By default, most browser's user agent stylesheet will style an ‹h1› with a large font size to make it look like a heading (although you could style it to look like anything you wanted). On the other hand, you could make any element look like a top level heading.

▶Consider the following

⇨This will render it to look like a top level heading, but it has no semantic value, so it will not get any extra benefits as described above. It is therefore a good idea to use the right HTML element for the right job.

⇨HTML should be ceded to represent the data that will be populated and not based on its default presentation styling.Presentatin (how it sholud look), is the sole responsibilty of CSS.

Some of the benefits from writing semantic markup are as follows:
  • Search engines will consider its contents as important keywords to influence the page's search rankingS (see SEO)
  • Screen readers can use it as a signpost to help visually impaired users navigate a page
  • Finding blocks of meaningful code is significantly easier than searching trough endless divs with or without semantic or namespaced classes
  • Suggests to the developer the type of data that will be populated
  • Semantic naming mirrors proper custom element/component naming

SEO

⭐ SEO (Search Engine Optimization) is the process of making a website more visible in search results, also termed improving search rankings.

⭐ Search engines crawl the web, following links from page to page, an index the content found. When you search, the search engine display the indexed content. Crawlers follows rules. If you follow those rules closly when doing SEO for a website, you give the site the best chances of showing up among the first results, increasing traffic and possibly revenue (for ecommerce and ads).

⭐ Search engines give some guidlines for SEO, but big search engines keep result ranking as trade secret. SEO combines official search engine guidlines, empirical knowledge, and theoretical knowledge from science papers or parents.
